Desarrollo de un Portal Web para la Materia IHC- seccion 1050

La interaccion humano-computadora

Hola Esto es una Prueba del Portal Desarrollado para la materia de Interaccion Humano Computador, en el cual hemos aprendidos las bases del Desarrollo Web, aplicado a las Intefaces Web, primeramente con el Software de Maquetacion Balsamiq, y luego llevando ese mismo proyecto a la programacion basica, HTML, CSS y Javascript, donde hemos aprendido las Bases Fundamentales de estos lenguajes y como con algunas simples lineas de codigos hemos podido crear una maravillosa Intefaz Web, la cual cuenta con los elementos mas importales para interaccion con el Humano.

Fundamentos del Desarrollo

Aprendimos los Fundamentos del Desarrollo Web, y con ello como hacer una simple interfaz que sea Resposiva, es decir con un Diseño Adactativo, que se ajusta al Dispositivo donde se Interactue con el, dando al usuario las satisfaccion de que su contenido es ajustable a su pantalla, con la cual podra navegar sin dificultap.

El Diseño Web Adactativo Responsive Web Design es un término cada vez más utilizado en el mundo del diseño web, se trata de, mediante el uso de CSS, adaptar un diseño a múltiples resoluciones de pantalla. Con ello podemos concluir que La interaccion humano-computadora: Es una disciplina que se encarga del estudio del intercambio de informacion mediante algun tipo de software y/o hardware entre los humanos y las computadoras, y cualquier Dispositivo que posea una conexion, con el internet o no, pero que Humano como usuario debera interactuar con el.

Tecnologias Vistas en el Proyecto

Balsamiq Mockups

Cuando queremos crear un proyecto nuevo, es normal que hagamos sketchs, bocetos o dibujos rápidos para tener una imagen visual de cómo será nuestra interfaz. Esto nos ayuda a más adelante poder crear el proyecto lo más semajante posible a nuesto boceto en papel. Mientras lo realizamos a lapiz, hacemos cambios, tachamos, tenemos mejor ideas y estos sketchs se convierten en el prototipo del proyecto final, en definitiva en Mockups. A este proceso se le denomina Wireframing que es la creación de la estructura de una interfaz..
